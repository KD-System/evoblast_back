## 1. Объект тестирования

**Наименование системы:** Evoblast - AI-ассистент с базой знаний

**Назначение:** Чат-бот для ответов на вопросы пользователей на основе загруженной документации с использованием технологии RAG (Retrieval-Augmented Generation).

### Состав системы:

1. Модуль загрузки и обработки документов
2. Модуль индексации (Vector Store)
3. Модуль чата с AI-ассистентом
4. Модуль генерации названий чатов
5. База знаний (MongoDB + GridFS)
6. Пользовательский интерфейс (веб-приложение)

### Технологический стек:

| Компонент | Технология |
|-----------|------------|
| Backend | FastAPI (Python 3.11) |
| База данных | MongoDB 7.0 + GridFS |
| Языковая модель | YandexGPT |
| Эмбеддинги | text-search-doc/query |
| Индексация | Yandex Cloud Vector Store (SearchIndex) |
| Инфраструктура | Docker, YandexCloud |
| Технология RAG | Yandex Cloud ML SDK |

**Функциональное тестирование**
- Загрузка документов различных форматов

- Отправка вопросов и получение ответов

- Работа с историей чатов

**Нагрузочное тестирование**

- Проверка времени отклика

- Релевантность ответов на основе базы знаний
